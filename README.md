# CX Standup Board

Real-time collaborative priority board for CX standup meetings.

## Features
- **Priority Board** — Drag & drop items across High (This Week), Medium (Next 2 Weeks), Low (This Month)
- **List View** — Filter by status, priority, and owner
- **Summary** — Overview by owner, priority, and status
- **Real-time sync** — Supabase realtime subscriptions
- **Owner filtering** — Sidebar filter by team member
- **9 Users** — Don, Mark, Kerushan, Daryl, Tunya, Tim, John, Greg, Verona

## Tech Stack
- Single HTML file (no build step)
- Tailwind CSS via CDN
- Supabase (Postgres + Realtime)
- Vercel hosting
